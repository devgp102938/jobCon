JobCon is a responsive job portal frontend application built using React. The project demonstrates component-based architecture, state-driven UI updates, client-side routing, and modern interface design principles.

This version represents the completed frontend phase. Backend integration is planned for a future MERN stack implementation.

Overview:

JobCon provides a structured and intuitive interface for browsing job listings. Users can filter opportunities dynamically, explore job details, and interact with a modal-based quick apply system.
The application focuses on clean UI design, reusable components, and responsive layout behavior across devices.
